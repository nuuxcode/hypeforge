# HypeForge - AI Compliment Generator

Turns a job title or person description into three wildly enthusiastic, slightly unhinged compliments.

## Features

- Three distinct compliments at once, forced by one persona from each bucket
- Each card has independent conversation history
- "Make it more dramatic" escalates only the selected card, using its own prior versions
- Copy to clipboard with per-card confirmation and fallback copy support
- Intentional loading, friendly errors, and mobile-first layout

## Tech Stack

Next.js App Router, TypeScript, Tailwind CSS, server API routes, Vercel AI SDK, Gemini via `@ai-sdk/google`, and a no-database HMAC cookie rate limit.

## Prompt Design

Each compliment is generated by a distinct comedic persona. The app picks one grand, one mythic, and one chaotic voice per run so outputs differ in structure, metaphor, and energy.

## Conversation State

Each card stores the current text, public compliment history, persona id, drama level, loading status, and copy state. Escalation sends only the public versions back to `/api/escalate`; the server rebuilds the persona/system prompt from `personaId`, so hidden instructions never ship to the browser.

## Environment

```bash
cp .env.example .env.local
```

Set `GEMINI_API_KEY` and `RATELIMIT_SECRET`. Keep `.env.local` out of git.

## Run Locally

```bash
pnpm install
pnpm dev
```

Open http://localhost:3000.
